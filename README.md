☁️ Cloud Storage Pro

A cloud-based file storage system built using Streamlit and Python that allows users to upload, manage, and access files through a simple web interface.

---

🚀 Features

- 🔐 Secure Login System
- 📤 Upload Files
- 📥 Download Files
- 👁 Image Preview
- ✏ Rename Files
- ❌ Delete Files
- 🔍 Search Files
- 📂 Filter (Images / Documents)
- 📊 Sort (Name, Size, Date)
- 📊 Dashboard with File Statistics
- ☁️ Cloud Deployment

---

🛠️ Tech Stack

- Python
- Streamlit
- OS Module
- Datetime Module

---

☁️ Deployment

This project is deployed using Streamlit Community Cloud.

Steps:

1. Upload project to GitHub
2. Connect repository to Streamlit Cloud
3. Deploy "app.py"
4. Access using generated URL

---

📂 Project Structure

Cloud-Storage-Pro/
│
├── app.py
├── requirements.txt
└── uploads/

---

▶️ How to Run Locally

1. Install dependencies:
   pip install -r requirements.txt

2. Run the app:
   streamlit run app.py
